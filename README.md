# Advanced building blocks - Enumerables
For this project, I implemented my own version of ruby's enumerable methods. 
Created: 
- #my_each
- #my_each_with_index 
- #my_select 
- #my_all? 
- #my_any?
- #my_none?
- #my_count
- #my_map
- #my_inject
